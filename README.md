# 📬 Newsletter Sign-Up Form

A responsive newsletter sign-up form built with HTML, CSS, and JavaScript.  
This project includes form validation using regex and a dynamic success message screen.

🔗 Live Site: https://bootcamp-final-project.netlify.app/

---

## ✨ Features

- Responsive layout for mobile and desktop
- Email validation using JavaScript (regex)
- Error message handling for invalid input
- Success message screen after form submission
- Dismiss button to return to form
- Clean UI based on Frontend Mentor design

---

## 🧠 What I Learned

While building this project, I practiced:

- DOM manipulation using JavaScript
- Form validation with regex
- Handling user events (`submit`, `click`)
- Showing and hiding UI sections dynamically
- Structuring a multi-screen layout in HTML
- CSS positioning and centering elements

---

## 🚀 How It Works

1. User enters an email address
2. JavaScript validates the email format
3. If invalid → error message appears
4. If valid → form hides and success screen appears
5. User can dismiss message and return to form
